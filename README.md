## ğŸ› ï¸ Installation

To run **Text-to-Speech-Generator** locally, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/govindchauhan-s/Text-to-Speech-Generator.git
    cd Text-to-Speech-Generator
    ```

2. **Serve the files locally:**  
   You can use Live Server in VS Code, or run a simple Python server:
    ```bash
    python -m http.server
    ```
   Then, open your browser and navigate to `http://localhost:8000/index.html`.

3. **(Optional) Install as PWA:**  
   Open the app in your browser and follow the prompt to install it on your device for offline access.
